Module Hierarchy

  • httxlib: HttxLib main module.
    • httxlib.httxauth: Authentication functions to answer to Basic and Digest HTTP authentication challenges
    • httxlib.httxauthcache: HttxLib authcache object to enable cached (or partially cached) authentication answer to authentication requests
    • httxlib.httxbase: Base Object for Connecting objects and primitives
    • httxlib.httxcertificates: Extensions of a HttxPassManager to be used to hold paths to certificate files or values for certificate validation
    • httxlib.httxcompression: Utility module to perform decompression of an http body
    • httxlib.httxcompressionset: HttxLib Set with locking semanctics to store the allowed compression values
    • httxlib.httxconnection: HttxConnection connecting object.
    • httxlib.httxcookiejar: Module implementing a deepcopyable Cookiejar (Cookiejar is already multithread safe)
    • httxlib.httxerror: Implementation of Exceptions raised by HttxLib
    • httxlib.httxmanager: Main connecting object HttxManager implementation
    • httxlib.httxnetlocation: Net Location connecting object HttxNetLocation implementation
    • httxlib.httxobject: Base httxlib object that ensures all objects receive a Lock
    • httxlib.httxoptions: Module that implements the domain wide options that connecting objects can share
    • httxlib.httxpassmanager: HttxLib password manager implementation
    • httxlib.httxrequest: urllib2 compatible Request object with additional functionality
    • httxlib.httxresponse: This module implements a series of hacks to the HTTPResponse class to enable extra functionaliy (and practical in some cases) for the HttxLib
    • httxlib.httxs: Reimplementation of HTTPSConnection to allow for certificate validation
    • httxlib.httxutil: Utility functions for http header parsing and digest authentication extracted from httplib (Python 2.6.5) for convenience.
